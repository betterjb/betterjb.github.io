### better way 5 시퀀스를 슬라이스하는 방법을 알자.

연습하기)

```python
$ a = ['a','b','c','d','e','f','g','h']

$ print(a[:]) # =>['a','b','c','d','e','f','g','h']

$ print(a[:5)] # => ['a','b','c','d','e']
  
$ print(a[:-1)] # => ['a', 'b', 'c', 'd', 'e', 'f', 'g']

$ print(a[4:]) # => ['e', 'f', 'g', 'h']

```


슬라이싱의 결과는 완전히 새로운 리스트이다. 
물론 원본 리스트에 들어 있는 객체에 대한 참조는 유지된다.
하지만 슬라이스한 결과를 수정해도 원본 리스트에 아무런 영향을 미치지 않는다.


```python
$ a = ['a','b','c','d','e','f','g','h']

$ b = a

$ c = a [:]

$ print(a)
['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h']

$ print(b)
['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h']

$ print(c)
['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h']

$ a.pop()
'h'

$ print(a)
['a', 'b', 'c', 'd', 'e', 'f', 'g']

$ print(b)
['a', 'b', 'c', 'd', 'e', 'f', 'g']

$ print(c)
['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h']
```
